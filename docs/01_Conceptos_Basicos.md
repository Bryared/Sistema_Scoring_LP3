# 2. Conceptos Básicos y Glosario

Este sistema es el **Cerebro Juez** de un banco digital.

### Glosario:
- **Onboarding:** Alta de nuevo cliente.
- **Scoring Crediticio:** Nota de confianza para préstamos.
- **DSR:** Carga Financiera. Qué porcentaje del sueldo gasta en deudas.
- **Compliance:** Cumplir las leyes anti-lavado y anti-terrorismo.
- **Smurfing:** Lavado rápido de dinero triangular.
- **XAI:** Inteligencia Artificial que explica sus decisiones.
- **Hash Criptográfico:** Sello de inmutabilidad para auditorías.
