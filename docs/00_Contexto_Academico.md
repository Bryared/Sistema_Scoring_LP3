# 1. Contexto Académico

**Institución:** Universidad Nacional Agraria La Molina (UNALM)
**Asignatura:** Lenguaje de Programación III (LP3)

El objetivo de este proyecto comenzó como una asignación para crear un Sistema Experto en Prolog, pero escaló hasta convertirse en un Core Bancario Corporativo Neuro-Simbólico con 11 módulos de riesgo, XAI, y una interfaz Streamlit interactiva.
