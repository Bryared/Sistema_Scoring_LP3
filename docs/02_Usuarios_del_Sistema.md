# 3. Usuarios del Sistema

Herramienta de **Back-Office** para empleados del banco:

1. **Oficial de Cumplimiento (Compliance Officer):** Evita multas detectando redes de lavado de dinero.
2. **Analista de Riesgos:** Evalúa los límites de deuda y toma decisiones manuales para políticos (PEP).
3. **Auditor Interno (SBS):** Vigila cobros indebidos de tasas y el riesgo de insolvencia de la cartera.
